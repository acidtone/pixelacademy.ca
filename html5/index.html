Some Rules/Conventions:

  • Use lowercase tags and attributes
  • Close tags that have content
  • Don't mismatch your closing tags
    • <strong><em>Yes</em></strong>
    • <strong><em>No</strong></em>
  • Wrap attribute values in single or double quotes
  • Indent nested elements for readability
  • Whitespace (spaces, new lines, tabs) collapses to a single space

Excercise 3

Turn on Slide URLs in index.html

reveal.js readme

  1. Scroll to Reveal.initialize in index.html
  2. Turn on URLs by setting history: true;
  3. Customize slide URL with <section> id attribute

Example: <section id="intro">

id Attribute

  • Can be added to any tag
  • Uniquely identifies an element within the document
  • May not contain any spaces
  • Avoid periods, colons and '#'
  • Main uses:
    • To point to styles in CSS
    • To hook into javascript code

Excercise 4

Add icon font to index.html

From Editor:

  1. Replace Slide Three with <i></i>
  2. Add icon font with class attribute

Example: <i class="fa fa-bath"></i>

class Attribute

  • Can be added to any tag
  • Separate multiple classes with a space
  • Individual class names (ex. fa-bath) can occur multiple times in a document
  • Main uses:
    • point to styles in CSS
    • hook into javascript code
