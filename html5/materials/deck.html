Some Rules/Conventions:

  • Use lowercase tags and attributes when possible
  • Indent child elements and long content for readability (whitespace generally doesn't matter)
  • Begin HTML5 documents with <!doctype html>
  • Close your tags (unless they're empty)
  • Wrap attribute values in single or double quotes
  • Link to CSS files using <link> not <style>
  • Link to javascript files using <script> (with closing tag) not <link>
