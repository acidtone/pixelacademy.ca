Laptop Nooner

Installing and Configuring Git

    Register for Nov 15

Industry Standard Source Code Management

In this lunch hour workshop we will help you install Git on your laptop, introduce you to the command line and walk you through the creation of your first repository.

Beginners Welcome

Git is a key skill to develop early in your career and it will be mandatory for most development teams and agencies. This workshop will get you through the initial hurdles of Git installation and configuration.

Command Line Comrads

The command line is scary. We'll give you a crash course in the commands you need to set up Git on your laptop. If there's time we'll show you the 4 or 5 'git' commands you'll use the most.
