Git Documentation

When in doubt read the docs. The Git homepage has an online reference, e-book, video tutorials and more.

Try Git

Try Git is a tutorial created by Code School and sponsored by GitHub. It's browser-based so you don't need to install Git to take the tutorial. You have to pass each step before you can proceed to the next.

  • Try Git

Download and Install

If you haven't installed Git on your machine, then now is the time! This page contains the install files for Mac, Linus and Windows. Download the file for your specific operating system, double-click and follow the instructions.

  • Download Git

GitHub Pages

GitHub is the most popular Git community and they have made it very simple to host your static website for free using GitHub Pages. You will need a GitHub account and a repository named username.github.io containing a static HTML website. Download a free responsive template from HTML5 Up.

  • GitHub Pages Tutorial
