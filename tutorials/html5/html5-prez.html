Setting up

  1. Install Chrome Browser
  2. Install modern text editor (Sublime, Atom)
  3. Download and unzip materials.zip:
    • pixelacademy.ca/tutorials/html5/materials.zip
  4. From Editor:
    1. File -> Open -> ../materials/template.html
    2. File -> Save As... -> my-project.html (or whatever)
  5. From Browser:
    1. File -> Open File... (Control-O) -> my-project.html

Some HTML5 Rules/Conventions:

  • Use lowercase text for HTML code
  • Close tags that have content
  • Don't mismatch your closing tags
    • <strong><em>Yes</em></strong>
    • <strong><em>No</strong></em>
  • Indent nested elements for readability
  • Whitespace (spaces, new lines, tabs) collapses to a single space
  • Wrap attribute values in single or double quotes
